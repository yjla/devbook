# 概览

这是一份**按 CodeTop 真实面试频率分档**的刷题清单，用于安排刷题优先级。分档依据是每题在 CodeTop 上被考的累计次数 (频度)，不是 Hot100 官方顺序、也不是主观判断。

:::tip
S 档 + A 档共 75 题的参考解法见 [S/A 档题解](./hot100-sa-solutions.md)，按题号升序排列。
:::

S/A/B 三档只装 **Hot100** 的题，按频度从高到低切：S 取前 25、A 取接下来 50、B 是剩下 25。「其他高频」单列**不在 Hot100**、但 CodeTop 频度 ≥ 100 的大厂常考题。

- **S 档**：Hot100 频度前 25 (频度 ≥ 197)，超高频，闭眼默写、白板 bug-free。
- **A 档**：Hot100 频度第 26–75 名 (频度 53–196)，高频，必须熟练。
- **B 档**：Hot100 里频度 &lt; 50 的 25 题，真实面试考得少，行有余力再刷。
- **其他高频**：不在 Hot100、但 CodeTop 频度 ≥ 100 的题 (手撕排序、链表变体、字符串、大数等)，按频度排序。

:::info
**数据口径**：CodeTop 全球综合榜 (全部公司)，按频度降序，数据截至约 2026-06。频度是累计被考次数，数字越大越高频。Hot100 共 100 题，频度 &lt; 50 的有 26 题 (最高 75 颜色分类 49，长尾到 &lt; 29)。精确到目标公司/岗位的频率，仍以 [CodeTop](https://codetop.cc/) 实时筛选为准。
:::

:::warning
**Hot100 ≠ 真实高频**。100 道 Hot100 里有 **26 题频度低于 50**——真实面试很少考 (如 347 前 K 个高频、84 柱状图、295 数据流中位数、763 划分字母区间)，全部沉在 B 档。反过来，CodeTop 高频前列有大量题**不在 Hot100** (手撕快排 353、88 合并两个有序数组 294、92 反转链表 II 269、103 锯齿形遍历 266)，见文末「其他高频」档。别只盯 Hot100，要和 CodeTop 取并集。
:::
