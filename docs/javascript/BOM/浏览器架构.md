# 浏览器架构

- 浏览器主进程：
  - 控制浏览器除标签页外的界面，包括地址栏、书签、前进后退按钮等
  - 负责各个页面的管理，创建和销毁其他进程
- GPU 进程
- 渲染进程（内核）：负责控制显示标签页内的所有内容，核心任务是将 HTML、CSS、JavaScript 转为用户可以与之交互的网页。默认情况下，每开启一个新的标签页，就会创建一个渲染进程
  - GUI 渲染线程：
    - 解析 HTML 和 CSS
    - 构建 DOM树 和 CSSOM树
    - 合成渲染树
    - 布局和绘制页面
    - 负责回流和重绘
  - JavaScript 引擎线程：和 GUI 渲染线程互斥，负责解析 Javascript 脚本，运行代码
  - 事件触发线程：控制事件循环。当 JavaScript 引擎线程遇到异步任务时，会将对应任务添加到事件触发线程中。当异步任务符合触发条件时，该线程就把异步任务添加到任务队列的队尾，等待 JavaScript 引擎线程处理。
  - 定时器线程：负责控制 `setTimeout` 和 `setInterval`，用于计时
  - 异步 HTTP 请求线程：负责将 `XMLHttpRequest` 请求的回调函数加入任务队列
- 网络进程
- 插件进程
