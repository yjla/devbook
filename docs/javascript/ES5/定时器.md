定时器回调执行完后timer是否存在。

取消后timeoutID是否存在



## 防抖

```js
const debounce = (fn, wait) => {
    let timer = null;
    return (...args) => {
        clearTimeout(timer);
        timer = setTimeout(() => {
            fn(...args);
        }, wait);
    }
}
```

## 节流

```js
const throttle = (fn, wait) => {
    let timer = lastArgs = null;
    return (...args) => {
        if (!timer) {
            fn(...args);
            timer = setTimeout(() => {
                if (lastArgs) fn(...lastArgs);
            }, wait);
        } else {
            lastArgs = args;
        }
    }
}
```

## sleep 函数

### 测试用例

```js
sleep(console.log, 1000, '打印回调');
```

### 实现

```js
const sleep = (callback, time, ...args) => new Promise(resolve => {
    setTimeout(() => {
        resolve(callback(...args));
    }, time);
});
```

## 用 setTimeout 实现 setInterval

### 测试用例

```js
_setInterval((...args) => {
    console.log(...args);
}, 500, 'test');
```

### 实现

```js
const _setInterval = (callback, timeout, ...args) => {
    let timer = null;
    const interval = () => {
        timer = setTimeout(() => {
            callback(...args);
            interval();
        }, timeout);
    }
    interval();
    return timer;
}
```

## 用 setInterval 实现 setTimeout

### 测试用例

```js
_setTimeout((...args) => {
    console.log(...args);
}, 500, 'test');
```

### 实现

```js
const _setTimeout = (callback, timeout, ...args) => {
    return setInterval(() => {
        callback(...args);
        clearInterval(timer);
    }, timeout);
}
```

## 间隔打印1到n的整数

### 测试用例

```js
intervalPrint(1000, 10);
```

### 实现

```js
const intervalPrint = (interval, n) => {
    let count = 0;
    const timer = setInterval(() => {
        console.log(++count);
        if (count === n) clearInterval(timer);
    }, interval);
}
```
