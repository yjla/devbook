# 类数组对象

类数组对象指的是拥有 `length` 属性和从零开始的整数索引属性的对象。类数组对象不是数组实例，因此不能调用数组原型链上的方法。



## 常见的类数组对象

- `arguments` 对象
- `NodeList` 对象



## 转换成数组

- `Array.from()`
- `slice()`
- 展开语法
