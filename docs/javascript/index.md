# JavaScript

按知识体系组织的 JavaScript 笔记，从语言底层到运行环境分为八大主题:

| 主题 | 内容 |
| --- | --- |
| **1 类型与内存** | 计算精度、内存管理、垃圾回收机制 |
| **2 执行机制** | 作用域、闭包、`this` 指向 |
| **3 原型与继承** | 原型链、继承的多种实现 |
| **4 异步编程** | 事件循环、`async/await` 原理、`requestAnimationFrame` 与 `requestIdleCallback` |
| **5 ES6+** | 箭头函数与普通函数、模块化 |
| **6 DOM** | 事件委托、`DOMContentLoaded` |
| **7 BOM** | 客户端存储、页面状态保存 |
| **8 浏览器原理** | 浏览器架构、渲染流程、回流重绘、从输入 URL 到页面呈现、V8 引擎 |
