# 架构

Node.js 可以分成三层：

- 最上层是 Node 标准库，提供各种内置 API
- 中间层是 node bindings， 用于 JavaScript 和 C/C++ 通信
- 最下层由 C/C++ 实现，包括：
  - V8 引擎
  - libuv：提供线程池、事件池、异步 I/O
  - C-ares：提供了异步处理 DNS 相关的能力。
  - http_parser、OpenSSL、zlib 等
