## 流程

1. 定时器：执行 `setTimeout()` 和 `setInterval()` 的回调函数。
2. 待定回调：执行延迟到下一个循环的 I/O 回调。
3. 空闲准备：仅 Node.js 系统内部使用。
4. 轮询：检索新的 I/O 事件，执行与 I/O 相关的回调。
5. 检测：执行 `setImmediate()` 回调函数。
6. 关闭的回调函数：一些关闭的回调函数，如 `socket.on('close', ...)`。
