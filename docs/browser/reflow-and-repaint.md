# 回流和重绘





### 如何减少回流？

改变文档布局流（flow）就会引发回流，回流必定导致重绘。而如果只是修改了 DOM 元素的样式，且没有影响几何属性的时候，就只会引发重绘（repaint）。

引起回流的常见因素：

- 元素的大小（包括边距，内容的改变，字体大小）或位置发生变化
- 添加或删除可见的元素
- 视口大小发生变化
- 激活伪类或改变 `class` 属性
- 读写 `offset`、`scroll` 和 `client` 类的属性
