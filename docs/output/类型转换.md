# 类型转换

## `{} + []`

```js
console.log({} + []);
```

结果是 `"[object Object]"`。

`+` 运算符会把两侧操作数转成原始值再运算。这里 `{}` 处于表达式中的运算位置，是**对象字面量**而非代码块，所以整个表达式是「对象加数组」。

- `{}` 转原始值：`({}).toString()` 得到 `"[object Object]"`
- `[]` 转原始值：`[].toString()` 得到 `""`
- 两个字符串相加：`"[object Object]" + ""` = `"[object Object]"`

:::warning
经典的坑在于 **语句开头** 的 `{}`。如果在 Chrome 控制台直接敲 `{} + []`，开头的 `{}` 会被解析成空代码块，于是只剩 `+[]`，结果是 `0`。一旦放进 `console.log(...)` 或赋值给变量，`{}` 就处于表达式位置，被当作对象字面量，结果才是 `"[object Object]"`。
:::
