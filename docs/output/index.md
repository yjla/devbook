# 输出题

这里收集 JavaScript 高频输出题，按主题分类整理。每道题都给出结论先行的解析，重点是搞清「为什么是这个结果」，而不是死记答案。

- [类型转换](./类型转换.md) —— 隐式转换、`+` 运算符的对象到原始值转换
- [`this` 指向](./this指向.md) —— 箭头函数、普通函数、独立调用的绑定规则
- [事件循环](./事件循环.md) —— 宏任务与微任务的执行顺序，浏览器与 Node 差异
- [作用域](./作用域.md) —— 变量提升、暂时性死区、参数默认值作用域
