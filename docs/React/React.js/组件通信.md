# 组件通信

- 父组件向子组件传值：`props`
- 子组件向父组件传值：父组件将一个回调函数作为 `prop` 传给子组件，子组件调用这个回调函数并通过参数的形式传值给父组件
- 跨级组件传值：`context`
- 子组件暴露方法给父组件：`ref` 转发
- 全局状态管理：Redux、Mobx
