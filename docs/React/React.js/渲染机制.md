# 渲染机制



## Props 不能直接绑定箭头函数

一旦给子组件的 props 直接绑定箭头函数，父组件的每次更新都会触发子组件的重新渲染。

在纯组件中，只要父组件重新渲染，React 就会比较子组件的 props 来决定是否更新子组件。而直接绑定箭头函数的写法相当于重新声明了一个函数，这个函数和原函数的内存地址不同，导致子组件的 props 发生了变化，因此子组件会重新渲染。

### 类组件

```js
class Child extends React.PureComponent {
  constructor(props) {
    super(props)
  }
  
  componentDidUpdate() {
    console.log('子组件更新了');
  }
  
  render() {
    return (
      <button onClick={this.props.handleClick}>{this.props.children}</button>
    )
  }
}

class Father extends React.PureComponent {
  constructor(props) {
    super(props)
    this.state = {
      count: 1
    }
  }
  
  render() {
    return(
      <>
        {this.state.count}
        <Child handleClick={() => {
            this.setState({
              count: this.state.count + 1
            });
          }}>按钮</Child>
      </>
    )
  }
}

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<Father/>);
```

解决办法：将箭头函数定义在组件上。

```js
class Father extends React.PureComponent {
  constructor(props) {
    super(props)
    this.state = {
      count: 1
    }
    this.handleClick = this.handleClick.bind(this);
  }
  
  handleClick() {
    this.setState({
      count: this.state.count + 1
    })}
    
  render() {
    return(
      <>
        {this.state.count}
        <Child handleClick={this.handleClick}>按钮</Child>
      </>
    )
  }
}
```

### 函数组件

```js
const { useState, useEffect } = React;

const Child = ({handleClick, children}) => {
  useEffect(() => {
    console.log('子组件渲染');
  });
  return (
    <button onClick={handleClick}>{children}</button>
  );
}

const Father = () => {
  const [count, setCount] = useState(1);
  return (
    <>
      {count}
      <Child handleClick={() => {
          setCount(count + 1);
        }}>按钮</Child>
    </>    
  );
}

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<Father />);
```

函数组件和类组件有所不同，每次重新渲染都会执行整个函数，无法保存状态。而使用 useCallback 会产生闭包陷阱，并不能解决这个问题。

```js
const Father = () => {
  const [count, setCount] = useState(1);
  const handleClick = useCallback(() => {
      setCount(count + 1);
  }, []);
  return (
    <>
      {count}
      <Child handleClick={handleClick}>按钮</Child>
    </>    
  );
}
```

解决办法：使用 React18 的 useEvent 或 ahooks 的 useMemoizedFn。

```js
const handleClick = useMemoizedFn(() => {
  setCount(count + 1);
});
```
