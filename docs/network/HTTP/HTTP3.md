# HTTP/3

HTTP/3 使用基于 UDP 的 QUIC 协议，具有以下特性：

- 快速握手
- 真正的多路复用
- 实现了类似 TCP 的流量控制、拥塞控制功能
- 集成 TLS 加密功能
- 连接迁移
