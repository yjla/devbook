# 第三方 Cookie

Cookie 与域（domain）关联。如果 Cookie 的域与用户所访问的页面的域相同，则该 Cookie 称为第一方 Cookie（ first-party cookie）；如果域不同，则它是第三方 Cookie（third-party cookie）。第三方 Cookie主要用于在网络上进行广告和跟踪。第三方 Cookie 只能由跨站响应的 `Set-Cookie` 设置。
