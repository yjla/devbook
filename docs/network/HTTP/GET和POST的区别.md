# GET 和 POST 的区别

- GET 用于获取数据，POST 用于提交数据。
- GET 有参数长度限制，POST没有。
- GET 的请求参数在 URL 中 ，POST 的请求参数再请求体中。
- 对服务器而言，GET 请求比 POST 请求更安全。
- GET 是幂等请求，POST 是非幂等请求
