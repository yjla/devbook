# 内容协商机制

- 请求头中的 `Accept` 字段列举了浏览器希望接收的资源类型，响应头中的 `Content-Type` 标识了服务器返回资源的类型。
- 请求头中的 `Accept-Charset` 字段用于告知服务器该浏览可以理解何种形式的字符编码，响应头中的 `Content-Type` 标识了服务器返回资源的字符编码。
- 请求头中的 `Accept-Encoding` 字段说明了浏览器可以接受的编码压缩形式，响应头中的 `Content-Encoding` 标识了服务器所采取的编码压缩形式。
- 请求头中的 `Accept-Language` 字段提示用户期望获得的自然语言的优先顺序，响应头中的 `Content-Language` 标识了服务器返回的自然语言顺序。
