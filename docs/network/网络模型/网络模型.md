# 网络模型

网络模型主要有以下两种：

- OSI 七层网络模型：物理层、数据链路层、网络层、传输层、会话层、表示层、应用层
- TCP/IP 四层网络模型：网络接口层、网络层、传输层、应用层

每一层的作用：

- 网络接口层：将数据封装成数据帧，发送到网络上，并以比特流的形式在介质上传输。
- 网络层：IP选址和路由选择。
- 传输层：建立和管理端到端的连接。
- 应用层：为应用程序提供服务。

每一层常见的的协议：

- 网络接口层：ARP
- 网络层：IP、ICMP。
- 传输层：TCP、UDP。
- 应用层：HTTP、DNS、FTP、TLS、SSH、SMTP
