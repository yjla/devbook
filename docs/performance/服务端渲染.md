# [页面卡顿，可能是哪些原因造成的？](https://www.cnblogs.com/manhuai/p/14630743.html)

- 先会检查是否是网络请求太多，导致数据返回较慢，可以适当做一些缓存
- 也有可能是某块资源的bundle太大，可以考虑拆分一下
- 然后排查一下js代码，是不是某处有过多循环导致占用主线程时间过长
- 浏览器某帧渲染的东西太多，导致的卡顿
- 在页面渲染过程中，可能有很多重复的重排重绘
- 内存泄漏
