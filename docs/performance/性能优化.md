# 性能优化

- 图片优化
  - 压缩图像
  - 使用响应式图像
  - 使用 WebP 格式的图像
  - 使用 CDN 托管图像
  - 延迟加载图像
- 字体优化
  - 预加载字体
  - 使用字体图标代替图片图标
- JavaScript 优化
  - 删除无用代码
  - 代码拆分
  - 模块化
  - Gzip 压缩
  - 使用动态 import 延迟加载 JavaScript
- CSS 优化
  - 延迟加载非必要 CSS
  - 尽量使用 CSS 动画
  - 降低 CSS 选择器的复杂度
- 网络优化
  - 使用 HTTP/2 协议
  - DNS 预解析
  - 预连接
  - HTTP 缓存
  - 避免多次重定向
  - 延迟加载（组件、路由、图像、代码）
- 渲染优化
  - 减少 DOM 元素的数量
  - 减少回流和重绘

- 事件优化
  - 防抖
  - 节流
  - 事件委托
- SEO 优化
  - 服务端渲染

- 体验优化
  - 骨架屏
  - 移动端滚动优化
  - 跳转动画
  - 渐进加载图片
