伪类与伪元素

一句话区分：伪类描述元素的某种状态或位置 (如 :hover、:nth-child)，匹配的是页面上已存在的元素；伪元素创建一个文档里本不存在的虚拟元素 (如 ::before、::after)，凭空生成可被样式化的内容。

本质区别​

维度伪类 :伪元素 ::
作用选中处于特定状态/位置的真实元素生成一个虚拟元素
操作对象DOM 中已存在的元素DOM 中不存在、由 CSS 创造
语法单冒号 :hover双冒号 ::before
典型代表:hover :focus :checked :nth-child() :not()::before ::after ::first-line ::first-letter ::placeholder
一个选择器中可用数量可多个一般只在末尾用一个
/* 伪类：鼠标悬停在这个真实的 a 上时 */
a:hover { color: red; }

/* 伪元素：在每个 li 内容前面凭空插入一个 "•" */
li::before { content: "• "; color: gray; }

为何伪元素用 ::、伪类用 :​

历史原因：CSS2 时代两者都用单冒号 (:before、:hover 都合法)。CSS3 为了在语法上区分 "选状态" 和 "造元素" 两个不同概念，规定伪元素改用双冒号 ::，伪类保持单冒号 :。

p:before  { content: "旧"; }   /* CSS2 写法，仍被浏览器兼容 */
p::before { content: "新"; }   /* CSS3 推荐写法 */
信息

为了向后兼容，现代浏览器仍然接受单冒号形式的伪元素 (如 :before)。但新代码应一律用双冒号 :: 写伪元素，让 "状态" 与 "虚拟元素" 在视觉上一眼可分。注意 IE8 只认单冒号，不过早已无需考虑。

::before / ::after 必须配 content​

这是最常踩的坑：没有 content 属性，伪元素就不会被渲染，哪怕你设了宽高背景也看不到。需要纯装饰、无文字时也要写 content: ""; (空字符串)。

/* 错误：缺 content，伪元素不生成 */
.box::before {
  width: 8px;
  height: 8px;
  background: red;
}

/* 正确：content 不能省 */
.box::before {
  content: "";       /* 必须有，哪怕是空串 */
  display: inline-block;
  width: 8px;
  height: 8px;
  background: red;
}

content 也可以放文字、attr 取值、计数器等：

.link::after  { content: " ↗"; }
a::after      { content: " (" attr(href) ")"; }  /* 取属性值 */
提示

::before/::after 默认是行内元素，生成的内容是 "元素内容区的最前 / 最后"，常用于图标、装饰线、清除浮动、角标等纯展示需求。注意它生成的内容无法被选中复制、不进入无障碍朗读，所以不要用它放真正重要的正文。

常用伪类清单​

结构性伪类 (按位置匹配)​

伪类含义
:first-child是父元素的第一个子元素
:last-child是父元素的最后一个子元素
:nth-child(n)第 n 个子元素，支持 2n (偶数)、odd、even 等表达式
:nth-of-type(n)同类型元素中的第 n 个
:only-child是父元素唯一的子元素
tr:nth-child(even) { background: #f5f5f5; }  /* 斑马纹表格 */
li:first-child     { font-weight: bold; }

状态伪类 (按交互/状态匹配)​

伪类含义
:hover鼠标悬停
:focus元素获得焦点
:active元素被激活 (鼠标按下未松开)
:checked单选/复选框被选中
:disabled / :enabled表单控件禁用 / 可用
:visited链接已被访问过
input:checked + label { color: green; }
button:disabled       { opacity: 0.5; cursor: not-allowed; }

否定伪类​

:not() 排除符合条件的元素：

li:not(.active)        { color: gray; }       /* 除 .active 外的 li */
input:not([type="hidden"]) { margin: 8px 0; }

对比表：常见伪类伪元素归类​

写法类别作用
:hover伪类悬停状态
:nth-child(2)伪类第 2 个子元素
:checked伪类选中状态
:not(.x)伪类排除 .x
::before伪元素内容前插入虚拟元素
::after伪元素内容后插入虚拟元素
::first-letter伪元素首字母 (做首字下沉)
::placeholder伪元素输入框占位文字样式
::selection伪元素被鼠标选中的文本片段
